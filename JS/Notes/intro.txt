                if(Condition){

            2.if-else :
                if(condition){

                    // statement

                if(condition 1){

                else if(condition 2){

                }

                This loops are used to check the condition at the starting of loop and run the statemenet

                        Syntax: 
                                inilization;
                                while(condition){

                                }
